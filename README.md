# Rick and Morty Characters

Мобильное приложение на Flutter для просмотра списка персонажей мультсериала «Рик и Морти». Загружает данные с [Rick and Morty API](https://rickandmortyapi.com/documentation/), позволяет добавлять персонажей в избранное и работает офлайн за счёт локального кеширования.

## Возможности

- **Список персонажей** — карточки с фото, именем, статусом, видом и локацией
- **Избранное** — сохранение в SQLite (Drift), сортировка по имени/статусу/виду, удаление свайпом
- **Пагинация** — подгрузка при скролле
- **Офлайн-режим** — кеширование загруженных данных
- **Тёмная тема** — переключение светлая/тёмная/системная
- **Локализация** — русский и английский

## Требования

- Flutter SDK ^3.11.0
- Dart ^3.11.0

## Сборка и запуск

```bash
# Установка зависимостей
flutter pub get

# Генерация кода (freezed, drift, slang)
dart run build_runner build --delete-conflicting-outputs

# Запуск
flutter run
```

Для сборки релиза:

```bash
flutter build apk    # Android
flutter build ios    # iOS
```
